# sleep-control
Easy toggle for Mac lids switching off my music :(


Preinstallation steps : Homebrew should be installed [https://brew.sh/]

Install via terminal : 
`brew tap dhr7v/sleepcontrol https://github.com/dhr7v/homebrew-sleep-control.git`
`brew install --cask dhr7v/sleep-control/sleep-control`

Security Warning : System Settings -> Privacy and Security
